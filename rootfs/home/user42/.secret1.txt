Lorsque vous tapez une commande, vous pouvez appuyer sur la touche Tab du clavier pour déclencher l’autocomplétion.

Par exemple, commencez à taper :

╭──────────────────────╮
│ cat s<Tab>           │
╰──────────────────────╯

Le shell devrait compléter votre commande en :

╭──────────────────────────────────────────╮
│ secret1.txt secret2.txt                  │
│ cat secret                               │
╰──────────────────────────────────────────╯

Si vous appuyez encore sur Tab, le shell ne complétera plus (il ne peut pas deviner le fichier voulu), mais il affichera les options possibles sur la première ligne : secret1.txt et secret2.txt.

Complétez un peu plus la commande en tapant `2` après `secret`, puis appuyez à nouveau sur Tab :

╭──────────────────────╮
│ cat secret2<Tab>     │
╰──────────────────────╯

Le shell complétera la commande en :

╭──────────────────────╮
│ cat secret2.txt      │
╰──────────────────────╯

Vous pouvez maintenant appuyer sur Entrée pour exécuter la commande.
Vous devriez voir le contenu du fichier `secret2.txt` s’afficher dans le terminal.
